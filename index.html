    Quiz Application

    This is a 10-question interactive quiz designed to exhibit the fundamentals of semantic HTML as well as CSS and JQuery. The user is invited to take a quiz consisting of 10 questions in which the user must identify a bird based on the image provided, selecting the correct answer from 4 possible choices. Upon hitting the "submit" button, the user will then be informed whether or not their selected answer is correct. The user will also be updated on their score throughout the duration of the quiz, and upon completion will be given the option to take the quiz again. This quiz app follows a11y practices.


    For more info please check out the live page or repo on this project.


    • HTML, CSS, Javascript, JQuery
    • Microsoft Visual Studio
    • Google Chrome Devtools
    • Github
